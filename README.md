# Nine6 CLI

Nine6 CLI is an opensource ai chatting program, and this program is CLI, so you need to run it on your terminal!.
This program is still in alpha version, bugs may occurred.

## Features

⦿ Code Highlight
⦿ AI generated text is formatted into human readable text
⦿ AI thinking animation

More Features Coming Soon.

## Usage

You can easily use this program by cloning this repository and run the .py file

```bash
git clone https://github.com/namatacha/nine6-cli
```

then you can cd to the cloned folder

```bash
cd nine6-cli
```

run the .py file

```bash
python main.py
```
